# TrainingForge AI

Initial Next.js, TypeScript, and Tailwind CSS project shell for an AI-assisted training plan generator.

## Tech Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Azure Static Web Apps workflow placeholder
- Azure Function API placeholder at `api/generate-training`

## Getting Started

Install dependencies:

```bash
npm install
```

Run the local development server:

```bash
npm run dev
```

Open `http://localhost:3000`.

## Project Structure

```text
app/                     Next.js pages and routes
components/              UI placeholders
lib/                     Shared API, markdown, and type helpers
api/generate-training/   Azure Function placeholder
public/                  Static assets
.github/workflows/       Azure Static Web Apps workflow placeholder
```

## Environment

Copy `.env.example` to `.env.local` and fill values when generation logic is implemented.
